飛騰雲端打卡

Online Mobile design

Project

此專案為大學光學任職期間之設計作品，為手機網頁版的優化項目，利用APP的方式呈現，讓使用者在操作上比較順手，此專案為剛接觸ＵＩ領域最具代表性的專案。

View Project

Problems

  • 所有功能過於制式統一，主要常用的功能反而被埋沒
  • Button按鈕選項普遍偏小，容易有誤觸的情形
  • 資訊量較大視覺上給人雜亂的感覺
  • 資料的輸入上，需要細心，避免誤植的情形，個人認為對於年 長的人，或是不常接觸APP資訊系統的人較不友善

Design

此App主要會遵守ios guideline ，採用 Flat Design的設計風格，讓畫面簡潔，把常用的功能改成四個主要分頁，命名上也比較簡潔希望讓用戶有乾淨舒適的使用體驗，也容易達到目的性。

View Project

打卡頁

  • Button精簡成一顆，系統設計上會直接抓出第一次點下去的時間
  • Beacon打卡收納在打卡頁面中

申請頁

首頁顯示簡易行事曆，可以馬上知道哪天可能有打卡異常，可以立即進行補假

簽核頁

有需要待審核的資料會逐筆秀出，直到全數審核完畢

個人頁

其他比較個人查詢的部分統一都會放在這邊，重要的項目會在前面的分頁秀出
